# SarvUday: Enterprise AI-Driven Mental Health Platform

## Overview
SarvUday is a comprehensive, multi-platform (Web and Mobile) mental health tracking and support system. It connects users seeking mental health guidance with qualified counsellors, leveraging AI-driven sentiment analysis to track emotional well-being and provide real-time insights.

## Core Stack
- **Backend**: FastAPI (Python), PostgreSQL, SQLAlchemy ORM, JWT Authentication.
- **Frontend (Web)**: React.js (Vite), Axios, Context API, CSS Modules.
- **Frontend (Mobile)**: React Native (Expo), Expo Router, Axios.
- **AI / ML**: PyTorch, Hugging Face Transformers (BERT), FastAPI microservice (`sentiment_model_api`).
- **Infrastructure**: Docker & Docker Compose.

## Vision
To build an accessible, highly secure, and deeply intelligent mental health platform that bridges the gap between individuals experiencing emotional distress and professional counsellors, acting as a proactive mental health companion.
