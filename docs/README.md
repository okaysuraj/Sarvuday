# SarvUday: AI-Driven Mental Health Platform

## Overview
SarvUday is a comprehensive web application for mental health tracking and support. It connects users seeking mental health guidance with qualified counsellors, leveraging AI-driven sentiment analysis to track emotional well-being and provide real-time insights, alongside a 24/7 AI cognitive behavioral therapy companion.

## Core Stack
- **Backend**: FastAPI (Python), SQLAlchemy ORM, JWT Authentication.
- **Frontend (Web)**: React.js (Vite), Axios, Context API, CSS Modules.
- **Databases**: Neon Cloud (PostgreSQL) and MongoDB Atlas (NoSQL Chat History).
- **AI / ML**: Hugging Face Spaces (hosting `model.gguf` via `llama-cpp-python` compatible server) for AI chatbot companion, and background sentiment analysis models.
- **Infrastructure**: Render (Backend hosting), Netlify (Frontend hosting).

## Vision
To build an accessible, highly secure, and deeply intelligent mental health platform that bridges the gap between individuals experiencing emotional distress and professional counsellors, acting as a proactive mental health companion.
